Jaeksoft Open Search Server is a search engine build in JAVA,
based on best JAVA technologies:

    * The lucene Text search engine library Apache Lucene
    * Stable and powerful server Apache Tomcat
    * An ergonomic user interface powered ZK direct RIA
 
Web site:
http://www.open-search-server.com

TO USE THIS APPLICATION :
you will need to have external libraries referenced in application bluid path:
 - commons-beanutils.jar
 - commons-codec-1.3.jar
 - commons-collections-3.2.jar
 - commons-digester-1.8.jar
 - commons-discovery-0.4.jar
 - commons-httpclient-3.1.jar
 - commons-logging-1.0.4.jar
 - jsf-api.jar
 - jsf-impl.jar
 - jstl.jar
 - junit-3.8.2.jar
 - log4j-1.2.11.jar
 - richfaces-impl-3.3.0.GA.jar
 - richfaces-api-3.3.0.GA.jar
 - richfaces-ui-3.3.0.GA.jar
 
 AND 
 - lucene-analyzers-2.4.1.patch579.jar
 - lucene-core-2.4.1-patch579.jar
 - lucene-core-2.4.1-patch579.jar
 - open-search-server.jar
