Provides for the view layer of the application. Thanks to JSF it is itself a MVC architecture. The model remains the same as in the application model layer {@link net.gisiinteractive.gipublish.model}. In some cases a decorator hides/extends the original pojo to trigger operations or provide additional computed information.
